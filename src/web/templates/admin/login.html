Default Badger Admin Login Page

Badger is an open source IoT RFID-based authentication, attendance tracking, notification, and rewards system for small to large scale events. Check the project out on GitHub.

This is the default website that comes with Badger. It contains the user login page, the user application panel, the admin login page (this page), and the admin management panel. When deploying Badger for your event, you should change and style these pages. What you are viewing are the placeholders containing code for all the necessary functionalities.

For more help, please consult the documentation.
